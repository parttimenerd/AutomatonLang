TODO Liste

Zur Aufnahme eines Kommentars in die TODO Liste einfach davor “TODO” schreiben und ein Leerzeichen zwischen beidem lassen, nun wird der Kommentar in die Liste aufgenommen wenn die Taste F5 zum Aktualisieren der Liste gedrückt wird.

Code Templates

Um ein Code Template zu verwenden geben sie zuerst das Schlüsselwort des jeweiligen Templates ein und drücken dann die Shift Taste.
